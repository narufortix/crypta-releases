Linux Essentials

Recurso de ejemplo del catálogo Crypta

1. La shell

La shell es la interfaz que te permite hablar con el sistema. La más común es bash.

2. Ficheros y directorios

Todo en Linux es un fichero. Aprende ls, cd, cp, mv, rm.

3. Permisos

Cada fichero tiene permisos para usuario, grupo y otros. Modifica con chmod.

4. Procesos

Usa ps, top, kill para ver y gestionar procesos.
